Python Project: Photo Sketch App

Project Overview: Developed a Python application that converts any given photo into a pencil sketch using edge detection techniques and image processing algorithms. Built with **Streamlit** to provide a simple, interactive web interface for users to upload their photos, generate pencil sketches, and download the result.

Tech Stack: Python, Streamlit, OpenCV, Jupyter Notebook, PyCharm

Key Features:

  • Interactive **Streamlit** interface for easy photo upload and sketch generation.
  • Edge detection algorithms (such as Canny and Laplacian) to generate realistic pencil sketches from photos.
  • Customizable sketching filters allowing users to modify the final output.
  • Use of image processing techniques like Gaussian blur to enhance the quality of the sketch.
  • **Download option** to save the generated sketch image to the user's local system.

Challenges Faced:

  • Fine-tuning the image processing algorithms to achieve a more realistic sketch effect.
  • Ensuring compatibility with various image formats and sizes while maintaining processing speed.

Future Improvements: Implementing machine learning models to predict the style of sketches based on the photo's content and adding more advanced filtering options like colorization of sketches.
